C# 笔记：Get 和 Set

發表於 2020-11-21 更新於 2020-11-22 分類於 C# 笔记

简介：

　　顾名思义，是记录 Get 和 Set 用法的 C# 笔记~
　　Get 和 Set 在《C# 程序设计教程（第3版）》中位于「5.7 属性」一节。

引用书中定义如下：

　　属性与字段相似，都是类成员，都有类型，可以被赋值和读取。属性提供了对类或对象成员的访问。通常情况下，将字段设置成私有的，设计一个对其进行读或写的属性。因此更充分地体现了对象的封装性。

语法格式为：

//public 可以换成 private 和 protected
public <数据类型> <属性名称> {
    get
    set
}

实例：

HB_Time.cs：

using System;
class HB_Time { //第5.7节 属性 Get 和 Set
    //核心数值，时分秒的赋值改变都会使其改变，时分秒的获取都要先获取它再处理。
    //它是个6位数，时占头2位，分占中间2位，秒占后2位。
    private int time; 
    public int Seconds {
        get { return time % 100; }
        set {
            while (value > 59) {
                value -= 60;
                time += 100;
            }
            time -= Seconds;
            time += value;
            if (Minutes > 59) time += 40;
        }
    }
    public int Minutes {
        get { return time % 10000 / 100; }
        set {
            while (value > 59) {
                value -= 60;
                time += 10000;
            }
            time -= Minutes * 100;
            time += value * 100;
            if (Minutes > 59) time += 4000;
        }
    }
    public int Hours {
        get { return time / 10000; }
        set {
            time -= Hours * 10000;
            time += value * 10000;
            while (Hours > 23) time -= 240000;
        }
    }
    public void Display() { //显示时间
        Console.WriteLine("时间为 [{0}:{1}:{2}]", Hours.ToString().PadLeft(2, '0'), Minutes.ToString().PadLeft(2, '0'), Seconds.ToString().PadLeft(2, '0'));
        return;
    }
}

Program.cs：

HB_Time t = new HB_Time {
    Seconds = 1, //系统直接调用 set 属性赋值
    Minutes = 35, //系统直接调用 set 属性赋值
    Hours = 12 //系统直接调用 set 属性赋值
};
t.Display(); //调用 Display() 方法显示时间
//time的值为123501
//输出：“时间为 [12:35:01]”
